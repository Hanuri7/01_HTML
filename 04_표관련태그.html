table

  1. table(표)

            - 웹 문서에서
          data를 정리하기 위해 사용

        - table 은 행, 열을 작성하기 위한
          영역(판)을 지정하는 태그

  2. tr(table row)

            - table의 한행을 나타내는 태그
      
  3.         - table에 출력될 data를
          작성하는 태그
        
        - 각 행의 열(col) 역할

  4. th(table header)

            - 열의 제목을 나타내는 태크
        - td와 같은 레벨로 작성
        - 굵은 글씨 + 가운데 정렬

  5. caption

    - 표 또는 이미지의 ...

  6. thead

    - table 상단 영역(값 작성 영역)

  7. tbody

    - table 중단 영역(값 작성 영역)

  8. tfoot

    - table 하단 영역(합계 작성 영역)

  9. colspan(열 병합)

    - 열로 나열 된 방향(가로, 옆) 병합

  10. rowspan(행 병합)

    - 행으로 나열 된 방향(세로, 위/아래) 병합

3,13,23,3
