form(형식, 양식)

    - 입력 양식을 작성하는 판(영역)
    - 내부에 작성된 값을 지정된 경로(html파일,서버)로 제출

    [속성]
    1. action(행동) : 제출될 파일/서버 경로 작성
    2. method(방식) : 제출 방식 지정
    3. name : 이름 지정(JS)
    4. target : 제출된 페이지를 현재 탭/새탭 어디서 볼지 결정

Button

    - 버튼을 만드는 태그
    - type에 따라서 용도가 달라짐

    type="submit"
    - form 내부에 작성된 경우
      해당 form의 모든 input 값을 action 경로로 제출

    type="reset"
    - form 내부에 작성된 경우
      해당 form의 모든 input 값을 Reset(초기화)

    type="button"
    - 기능 없음
      JS를 이용해 사용
