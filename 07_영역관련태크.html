inline 형식 요소

    - 화면 수평 분할
      (가로 화면/영역을 잘라서 열로 분할)

    - width / heigth 크기 지정 속성 적용 X
      작성된 내용 만큼 크기만 차지함

    - inline 형식 요소 종류
      b, i, strong, em, mark, span(영역 분할)
inline 형식은 행을 잘라서 열을 만드는 성질을 가지고 있고 줄이 바뀌지 않고 계속 옆으로 작성 (inline 형식 요소는 글자 취급)

block vs inline : 영역 지정 방식의 차이

    - block : 항상 직사각형의 형태로 영역을 지정

    - inline : 작성된 내용(content) 단위로 영역을 지정
