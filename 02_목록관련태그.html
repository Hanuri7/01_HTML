선대경의 페이지

ul(Unordered List) : 순서가 없는 목록

li(List Item) : 목록의 한 줄

  • 햄버거
  • 김밥
    • 참치
    • 치즈
    • 고추
      • 오이
      • 꽈리
      • 청양
  • 라면
  • 샌드위치
  • 국밥

ol(Ordered List) : 순서가 있는 목록

  1. 우육면
  2. 김밥
  3. 버거
      * type : 표시되는 숫자 형식 지정 
        type="a" : 영어 소문자(a,b,c)
        type="A" : 영어 대문자(A,B,C)
        type="i" : 로마 소문자(i,ii,iii)
        type="I" : 로마 대문자(I,II,III)
        type="1" : 숫자(1,2,3)(기본값)

      * start : 목록 시작 번호
      * reversed : 역순
  1. HTML5
  2. CSS3
  3. JavaScript
  4. Java
  5. DB(Oracle)

Emmet : HTML, CSS 빠른 자동완성 지원 플러그인

      > : 태그 내부에 작성(하위 생성)
      + : 태그 다음에 작성(형제)
      *숫자 : 숫자 만큼 생성
      {} : 태그 내부에 작성될 내용
      $ : 1부터 1씩 증가(시퀀스)
  1. 아이템1
  2. 아이템2
  3. 아이템3
  4. 아이템4
