4. p(Paragraph, 문단,단락)

글을 작성하는 용도의 태그

pre 처럼 형식이 유지되진 않음

글의 주제가 바뀔 때 마다 p를 새로 작성
문단(p) 사이에 여백이 생성

글자의 적용 되는 태그

b(bold) : 단순글자를 굵게 표시
i(italic) : 단순글자를 기울여 표시
strong : 글자(단어)를 강조
em : 문장의 맥락을 변화시키는 강조 
      * 웹 접근성이란?
      - 일반인 뿐만 아니라
        장애인, 고령자들을 위해
        웹 사이트에서 제공하는 모든 서비스를
        이용할 수 있도록 보장하는 것(의무사항)
    
      mark : 형광펜 효과주는
      u(under) : 밑줄 긋는
      s : 글자 가운데 선을 긋는 (취소, 오타)
      del : 삭제선을 긋는 (있다 없어짐)

      * s, del 용도에 맞게 사용
      - 관리자 명단 : A(탈퇴),B,C(오타)  

      sup(윗첨자) : x3
      sub(아랫첨자) : log2

      abbr(abbreviation, 약어) : 마우스 오버시 툴팁 제공
        HTTP 
        HTML
