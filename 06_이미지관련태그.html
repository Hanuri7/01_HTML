img

    - 웹 문서에서 이미지를 삽입하는 용도
    [속성]
    (속성 : 시작 태그에 작성하는 내용)

    1.src : 삽입할 이미지의 경로를 작성하는 속성(파일 위치, url)
    2.width / height : 너비 높이를 지정하는 속성(단위 : px, % em, rem 등) 
    3.alt : 이미지 설명을 작성하는 속성

img 기본 사용법

img size 지정(width / height)

고정 size 단위

    - 화면의 크기가 변해도 항상 고정된 크기를 지님
    - 단위 : px

가변 size 단위

    - 브라우저 크기 또는 지정된 영역의 크기에 대한
      상대적인 크기 단위
    - 단위 : %, em, rem 등
